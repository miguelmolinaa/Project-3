This JavaScript code loads data from a JSON file containing property listings and creates interactive data visualizations using the Plotly library. It includes the following functionalities:

State Pie Chart: It generates a pie chart displaying property counts by state, and it allows users to interactively select a state from a dropdown menu to see property type distribution for that state.

Property Type Counts: It counts and displays the number of properties for each property type and updates the counts dynamically based on the selected state.

Specialist Sales Bar Chart: It calculates and displays a bar chart showing the number of houses sold and total sales for different specialists involved in property sales.
